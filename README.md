# 🔢 Matrix Operations Tool

A Python-based matrix calculator built using NumPy that allows users to perform common linear algebra operations through an interactive command-line interface.

## ✨ Features

* Matrix Addition
* Matrix Subtraction
* Matrix Multiplication
* Matrix Transpose
* Determinant Calculation
* User-defined matrix input
* Input validation for matrix dimensions

## 🛠 Tech Stack

* Python 3
* NumPy

## 📂 Project Structure

```text
matrix-operations-tool/
├── matrix_tool.py
├── requirements.txt
└── README.md
```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/ianusha08/matrix-operations-tool.git
cd matrix-operations-tool
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python matrix_tool.py
```

## 📋 Supported Operations

| Operation      | Description                               |
| -------------- | ----------------------------------------- |
| Addition       | Adds two matrices                         |
| Subtraction    | Subtracts one matrix from another         |
| Multiplication | Performs matrix multiplication            |
| Transpose      | Returns the transpose of a matrix         |
| Determinant    | Calculates determinant of square matrices |
